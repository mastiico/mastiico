## 👨‍💻 Sobre Mim

Sou um **Desenvolvedor de Software Backend Pleno** apaixonado por arquiteturas limpas, microsserviços e sistemas de alta performance. Apesar de minha maior especialidade ser o Backend, possuo um perfil multidisciplinar: atuo fortemente com desenvolvimento **Frontend Moderno** e engenharia de **Automação/RPA**.

🔹 **Backend & Arquitetura:** Criação de APIs robustas e escaláveis, integração com serviços cloud e mensageria (Filas/Cache).  
🔹 **Ecossistema Frontend:** Experiência com interfaces dinâmicas utilizando React, TypeScript, Tailwind e ecossistema Vite.  
🔹 **RPA & Web Scraping:** Desenvolvimento de crawlers complexos e automações para extração de dados e ganhos de eficiência.  
🔹 **Sempre em evolução:** Explorando constantemente novas tecnologias para agregar valor aos produtos em que atuo.
